# cobots
First install rust.
```console
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
Then install dependencies.
```console
sudo apt install libclang-dev ros-galactic-xacro
```

Make install script executable
```console
chmod +x install.bash
```
Run the installer.
```console
./install.bash
```
